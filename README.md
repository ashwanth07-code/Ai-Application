# AI Applications Platform

## 🌟 Overview
A comprehensive AI-powered platform featuring four intelligent applications:
- 🤖 **AI ChatBot** - Conversational AI assistant
- 📧 **Spam Filter** - Email spam detection system
- 🎤 **Speech Recognition** - Real-time voice to text conversion
- ❓ **Q&A System** - Document-based question answering

## 🏗️ Architecture
- **Frontend**: React.js with Material-UI
- **Backend**: Node.js/Express
- **AI Models**: Python with Transformers, scikit-learn
- **Deployment**: Docker, Nginx

## 📋 Prerequisites
- Node.js (v18+)
- Python (3.8+)
- npm or yarn
- pip (Python package manager)

## 🚀 Quick Start

### Method 1: Manual Setup
```bash
# 1. Clone the repository
git clone <repository-url>
cd ai-applications-platform

# 2. Install dependencies
make install-all

# 3. Start the application
make run-all
