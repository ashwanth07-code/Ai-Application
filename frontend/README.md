# AI Applications Platform - Frontend

## 📋 Overview
This is the React.js frontend for the AI Applications Platform, providing a modern, responsive interface for four AI-powered applications.

## 🚀 Features

### 1. 🤖 AI ChatBot
- Real-time conversational AI
- Voice input support
- Message history
- Typing indicators
- Responsive design

### 2. 📧 Spam Filter
- Email content analysis
- File upload support (.txt, .eml)
- Real-time spam detection
- Confidence scoring
- Check history

### 3. 🎤 Speech Recognition
- Real-time audio recording
- Multiple recognition engines
- Confidence visualization
- Transcription history
- Export to text file

### 4. ❓ Q&A System
- Context-based question answering
- Multiple document support
- Answer confidence scoring
- Question history
- Copy answers to clipboard

## 🛠️ Technology Stack

- **React 18** - UI library
- **Material-UI v5** - Component library
- **React Router v6** - Navigation
- **Axios** - API requests
- **RecordRTC** - Audio recording
- **Web Speech API** - Speech recognition
